创建文章

方式一

在 cmd 中输入hugo new content post/文章名称/index.md 即可生成对应文件，文件头部会自动生成类似下方图片的参数设置，通过修改相关参数即可使该文章在主页中出现不同的效果，同时还可以给文章起到分类作用。参数设置成功后即可输入文章相关内容。

注意！参数上边与下边都需要用分割线分隔开

默认无法生成归类等参数的设置，可以通过源代码模式编写即可，如下

方式二

在路径 content/post 下创建以文章名命名的文件夹，在该文件夹下创建 index.md 文件，在文件中添加 yaml 代码，按如下方式设置参数即可

创建分类

找到路径 content/categories 在该文件下创建如下文件夹和文件 分类名称/_index.md , 在 .md 文件的头部输入如下类似的参数即可成功创建一个分类。

注意！参数必须写在 YAML Front Matter 中

1

---
title: "Test"
description: "This is an example category"
slug: "test"
image: "hutomo-abrianto-l2jk-uxb1BY-unsplash.jpg"
style:
    background: "#2a9d8f"
    color: "#fff"
---
  • 参数介绍：
    • title：分类标题
    • description：分类描述，会在归档中显示
    • slug：在右侧边栏显示分类名称

添加友链

按照如下格式，在 content/page/links/index.md 下的 yaml 代码中添加即可:

    website: https://github.com
    image: https://github.githubassets.com/images/modules/logos_page/GitHub-Mark.png
  - title: TypeScript
    description: TypeScript is a typed superset of JavaScript that compiles to plain JavaScript.
    website: https://www.typescriptlang.org
    image: ts-logo-128.jpg

image 接受本地文件和线上文件

将文章同步到 github 上

直接通过 git 将包含 hugo 执行程序和该主题所有相关配置文件的文件夹重新上传至仓库中即可，github 将通过 action 自动同步静态网页文件到公开为站点的仓库中，文章就成功的上传到博客中了。
