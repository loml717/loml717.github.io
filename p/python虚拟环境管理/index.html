Python虚拟环境管理

什么是虚拟环境

  • python环境

    • Lib 用于放置 python 可调用的库，其中包含标准库和 site-pakages。site-pakages 用于存放下载的第三方库。
    • Scripts 用于存放可执行文件
    • python.exe 即为 python 解释器，用于运行 py 文件

  • 虚拟环境

    虚拟环境实际上就是 python 文件夹的复制品，但与原环境有 3 点不同

    • 标准库不再被复制，直接调用原环境的标准库。site-pakages 文件夹里为空。
    • 虚拟环境应该有自己的名称
    • python.exe 被移到到 Scripts 中

venv 基本使用

创建名为 venvdemo 的虚拟环境：python -m venv venvdemo

**启动虚拟环境：**切换到文件目录 \venvdemo\Scripts 下，在 cmd 或 powershell 中输入 activate 即可激活，此时路径前会出现这样的字样 (venvdemo) ，说明激活成功

保存和复制虚拟环境

输出环境中所包含的所有第三方包：pip freeze

通过命令 pip freeze > requirements.txt 将环境的所有包信息保存到文件中，即可完美保存环境

**恢复环境：**创建好虚拟环境后，将 requirements.txt 文件中的所有包安装至环境即可，pip install -r requirements.txt
